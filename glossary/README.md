# Universe Crystal Glossary

This directory contains the official terminology and concept definitions of the **Universe Crystal** framework.

These documents establish the official language used throughout all Universe Crystal publications and ensure conceptual consistency across different editions and future research.

## Contents

### Official Glossary of Universe Crystal

The **Official Glossary of Universe Crystal** provides the official definitions of the core concepts used throughout the framework.

Each concept is defined only once.

The glossary defines **what each concept means**, but does not provide theoretical arguments or detailed explanations.

Those discussions belong to the **Universe Crystal Research Report**.

### Official Terminology of Universe Crystal

The **Official Terminology of Universe Crystal** standardizes the official English terminology used throughout the project.

It serves as the reference for maintaining consistent terminology across all official documents.

## Relationship to Other Documents

The glossary is one component of the official Universe Crystal documentation system.

* **Official Terminology** — Defines the official terminology.
* **Official Glossary** — Defines the official meanings of concepts.
* **Canonical Baseline** — Defines the official philosophical framework.
* **Research Report** — Presents the complete explanation of the framework.

## Version Policy

The Official Glossary and Official Terminology evolve together with the Universe Crystal framework.

New concepts or revised definitions are introduced only through new official versions following updates to the Canonical Baseline.
